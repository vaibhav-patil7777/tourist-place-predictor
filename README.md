An AI-powered web application that recommends tourist places in India based on user preferences such as weather, crowd level, tourism type, and budget.

---

## Features

-  **User Authentication**
  - Secure Signup & Login system
  - Password hashing using Werkzeug
  - Role-based access (Admin / User)

-  **Machine Learning Recommendation**
  - Uses Nearest Neighbors algorithm
  - Recommends best tourist places based on user preferences

-  **Google Maps Integration**
  - Embedded maps for each recommended place
  - Direct ‚ÄúOpen in Google Maps‚Äù option for navigation

-  **Tourism Chatbot**
  - Provides instant suggestions for beach, hill, and religious tourism

-  **Premium UI**
  - Clean and responsive design
  - Half blue / half white modern layout for recommendation page

---

##  Machine Learning Details

- **Algorithm:** Nearest Neighbors (KNN-based recommendation)
- **Preprocessing:** Label Encoding for categorical features
- **Features Used:**
  - Weather
  - Crowd Level
  - Tourism Type
  - Budget Level
